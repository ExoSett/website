A building system for modular accommodation

A robust framework for change.

ExoSett separates the long-lived structural frame of a building from the accommodation it hosts.

Removable accommodation modules are supported within an independent exoskeletal frame. Modules can be installed, removed or replaced as needs change, without relying on neighbouring modules for structural support.

Development

A system under active development

ExoSett is a developing building-system concept. Its architecture, components and potential applications are being defined alongside independent structural engineering investigation.

This website documents the system as it develops. It distinguishes between the principles that define ExoSett, current design work and illustrative stories about how the system might be used.
