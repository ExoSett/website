Cell

A cell is a defined space within an accommodation frame that can hold one accommodation module.

The cell remains part of the frame’s structure and layout even when the module occupying it is installed, removed or replaced.

How it works

Each cell is defined by the surrounding frame nodes, columns and beams. It provides the space, support points and clearances needed to install and secure a compatible accommodation module.

One occupied cell contains one module. Larger homes, rooms or other spaces can be created by connecting modules installed in adjacent cells, while each module remains independently supported within its own cell.

A cell may also remain empty during construction, module replacement or a period of reduced demand. The accommodation frame continues to support the rest of the building, while the empty cell is safely protected until it is used again.

Because the cell identifies a stable location rather than a particular physical module, it can also provide a consistent address for installation, services, inspection and maintenance records.
