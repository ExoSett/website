Service interface

A service interface connects an accommodation module or shared facility to the building’s service routes.

It provides defined connection points for utilities such as water, waste, power, data, ventilation and other building services.

How it works

The interface separates the module’s internal systems from the building-wide distribution systems. This allows either side to be inspected, maintained or altered without unnecessary disturbance to the other.

Depending upon the services involved, the interface may include pipes, sockets, couplings, valves, connectors, meters, isolation devices or control equipment.

Service interfaces may be standardised for particular module types or building uses. Clear connection arrangements can simplify installation, reduce errors and allow modules to be exchanged without extensive modification to the main service routes.

The interface must remain safe and accessible while accommodating installation tolerances, movement and the different maintenance requirements of each service.

By concentrating service connections at defined locations, ExoSett makes it easier to add, remove, replace or reconfigure modules as the building changes.
