Installation frame

The installation frame is temporary equipment used to insert and remove accommodation modules.

It docks with the accommodation frame and provides a stable, aligned transfer path between external handling equipment and the target cell.

How it works

The installation frame is positioned at the required cell and secured to defined docking points on the accommodation frame. It is then aligned with the cell so that the module can move safely into or out of the building.

During installation, a crane, hoist, vehicle or other handling system brings the module to the installation frame. The frame supports and guides the module until it reaches its final position and is secured within the accommodation frame.

During removal, the process is reversed. The module is disconnected from access, utilities, supports and restraints before being transferred onto the installation frame and passed to the external handling equipment.

The installation frame may use rails, rollers, bogies, skids or powered mechanisms to control movement. Once the operation is complete, it can be detached and moved to another cell or another compatible ExoSett building.
